_.join

_.join(array, [separator=','])

官方解释：

Converts all elements in array into a string separated by separator.

释义：

将数组中的所有元素转换为用分隔符分隔的字符串。

理解：

将数组转成字符串

Since：4.0.0

参数：

array(Array): 要转换的数组。

[separator=','](String): 每个元素的分离器。

返回值：

(string): 返回转换后的字符串

例子：

_.join(['a', 'b', 'c'], '~');
// => 'a~b~c'
