_.fill

_.fill(array, value, [start=0], [end=array.length])

官方解释：

Fills elements of array with value from start up to, but not including, end.

释义：

从开始到结尾(但不包括end)填充数组元素。

理解：

这个方法会改变数组，可以为数组填充默认值。

Since：3.2.0

参数：

array[Array]: 要查询的数组。

[value](*): 要填充数组的值。

[start=0](number): 起始位置

[end=array.length](number): 结束位置

返回值：

(Array): 返回填充后的数组

例子：

var array = [1, 2, 3];

_.fill(array, 'a');
console.log(array);
// => ['a', 'a', 'a']

_.fill(Array(3), 2);
// => [2, 2, 2]

_.fill([4, 6, 8, 10], '*', 1, 3);
// => [4, '*', '*', 10]
